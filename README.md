## 주요 레포지토리 구성

```
/books.csv               // 데모를 위한 예시 도서 데이터
/create_table.sql        // 테이블을 만드는 SQL 스크립트
/insert.sql              // 테이블에 예시 데이터를 집어넣는 SQL 스크립트
```
